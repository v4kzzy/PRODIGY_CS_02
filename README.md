# PixelLock - Image Encryption Tool

A secure tool to encrypt images using "v4kzy" (text-based) keys.

## Features
- **Modern GUI:** Dark mode interface.
- **Easy Keys:** Use names (like "v4kzy") instead of numbers.
- **Dual Security:** Uses XOR math + Pixel Swapping.

## How to use
1. Run `PRODIGY_WD_02.py`.
2. Select an image.
3. Enter a secret key name.
4. Click Encrypt.
